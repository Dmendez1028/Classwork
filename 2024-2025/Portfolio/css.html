Cascading Style Sheets (CSS)

CSS is a computer language for laying out and structuring web pages (HTML or XML).

Css cascade

styling rules “cascade” down from several sources.

CSS inheritance

controls what happens when no value is specified for a property on an element

CSS source order

the underlying mark-up that users rely on to navigate and access content.

CSS specificity

the algorithm used by browsers to determine the CSS declaration that is the most relevant to an element, which in turn, determines the property value to apply to the element

CSS rule

a grouping of one or more CSS properties which are to be applied to one or more target HTML elements

CSS declaration

the key-value pair of a CSS property and its value.

CSS property

used to set the style or assign behaviour to HTML elements.

CSS value

set against CSS Properties and reside within CSS declaration block, which is a part of the CSS rule / statement.

CSS selector

a pattern of elements and other terms that tell the browser which HTML elements should be selected to have the CSS property values inside the rule applied to them

id selector

uses the id attribute of an HTML element to select a specific element.

class selector

selects elements with a specific class attribute
