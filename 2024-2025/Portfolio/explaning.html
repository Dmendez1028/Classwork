Vocab about forms

HTML Method

Specifies how to send from data. The form data is sent to a specific page from the action tag.

HTML Legand

The legand element is used in HTML forms to provide a caption or a title for the associated fieldset element.

HTML Fieldset

The fieldset tag in html is used to make a group of related elements in the form

HTML Form Action

Specifies where to send the from-data when a form is submitted

HTML Raido Buttons

The HTML input type defines a radio button, grouped by sharing the same name attribute.

HTML Textarea

HTML textarea is a element that is also used often as a form to collect user inputs and comments or reviews.

HTML Button

HTML buttons is a tag that is clickable and that the user can use, a button could allow the user to submit such as forms, etc.

HTML Input Tag

The html input tag defines the field where the user can enter the data. The type attribute determines what type of user input is taken.

HTML Attributes

Html attributes helps modify and control html elements.

HTML Form

The form element is a container for different types of input elements, such as text fields, checkboxes,and etc

HTML Label

The label elements is used to associate a text label with a form input field. the label is used to tell users the value that should be entered in the associated inout field.
