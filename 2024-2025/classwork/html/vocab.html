layer

a distinct level within a hierarchical structure that separates different functions or responsibilities within a system.

user interface (UI)

the point of contact between humans and computers

software

a set of instructions, data or programs used to operate computers and execute specific tasks

application

a computer program designed to help people perform an activity

user-interface designer

someone who designs the graphical user interface of an app, website, or device that a human interacts with

front-end developer

creates websites and applications using web languages such as HTML, CSS, and JavaScript that allow users to access and interact with the site or app

back-end developer

build code that allows a database and an application to communicate with one another.

database developer

responsible for the design, programming, construction, and implementation of new databases, as well as modifying existing databases for platform updates and changes in user needs.

a software design principle that involves breaking a computer program into distinct sections, or modules, that each address a separate concern
