<asie> tag a semantic element introduced to mark up content that is indirectly related to the main article content of a page.
<article> tag represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable
<section> tag defines the section of documents such as chapters, headers, footers, or any other sections
<figure> tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc
<figure caption> tag defines a caption for a <figure> element
<video embed> tag where you add the video directly to your website, versus sending visitors to a different page to view it.
<audio embed> tag an inline element that is used to embed sound files into a web page
<video and audio source> tag The video tag is used to embed video content in a document, such as a movie clip or other video streams. The audio tag is used to embed sound content in a document, such as music or other audio streams.
<meta tag for charset> tag an attribute that tells your browser what character encoding to use for this webpage to show the content as desired
<meta tag for keywords> tag help tell search engines what the topic of the page is.
<meta tag for viewport> tag tells the browser that the width of the screen should be considered the "Full Width" of the page
<meta tag for refresh> tag a method of instructing a web browser to automatically refresh the current web page or frame after a given time interval
