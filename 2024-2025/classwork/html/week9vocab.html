Vocab about JavaScript

Number.parseFloat()

used to accept the string and convert it into a floating-point number

document.getElementById()

returns an Element object representing the element whose id property matches the specified string.

const

a keyword that defines a variable or pointer as unchangeable

document.querySelectorAll()

a method in JavaScript used to select a group of elements in the Document Object Model (DOM) based on a CSS selector.

document.getElementsByName()

a method in JavaScript that allows you to select all elements in the DOM with a specific name attribute

Number.parseInt()

parses a value as a string and returns the first integer.

typeof

an operator provided by several programming languages to determine the data type of a variable.

let

associates a function definition with a restricted scope.

Array.from()

type of global object used to store data

document.getElementsByClassname()

returns an array-like object of all child elements which have all of the given class name(s).

document.addEventListener()

allows you to add event listeners on any HTML DOM object such as HTML elements, the HTML document, the window object, or other objects that support events, like the xmlHttpRequest object.

7 JavaScript Primitives types

the simplest forms of data, representing a single value

ol

represents an ordered list of items
